\# Music Collection Manager



Desktop application for managing a music collection developed using C#, WPF and MySQL.



\## Features



\* User authentication and registration

\* Role-based access (Administrator / User)

\* Album and track management

\* Search and filtering functionality

\* Data storage using MySQL database

\* Report generation



\## Technologies



\* C#

\* .NET

\* WPF

\* MySQL



\## Project Structure



\* Music\_collection – main application source code



\## How to Run



1\. Clone the repository.

2\. Open Music\_collection.sln in Visual Studio.

3\. Configure the MySQL database connection.

4\. Build and run the application.
